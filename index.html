Sobre el Proyecto

Este proyecto consiste en una plataforma web desarrollada como parte del Trabajo Final de la asignatura Ingeniería Web. La aplicación permite gestionar usuarios, contenidos dinámicos, y fue desarrollada usando HTML, CSS, JavaScript y tecnologías backend como Node.js y MongoDB.

Puedes ver el código fuente completo en nuestro repositorio en GitHub.
